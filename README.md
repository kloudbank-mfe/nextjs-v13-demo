## Getting Started

nextjs v13, pages directory 기반 module federation 적용

- yarn 기본 create next-app 활용하여 host, remote app. 각각 생성
```sh
yarn create next-app ...
```

# host + remote 실행 방법

- host: port 3000
- remote: port 3001

```sh
# 1. install
$ yarn 
or 
$ yarn install --force

# 2. start
$ yarn start
```
